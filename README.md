# Build Rotation process

## Usage

```
Usage:
  build-rotate [OPTIONS] [ARGS]

Options:
  -e, --emulate BOOL     Emulate process without real execution
  -k, --keep [NUMBER]    Max numbers of builds to keep (Default is 4)
  -b, --build NUMBER     Build number
  -f, --folder [STRING]  PWD (Default is current working directory)
  -h, --help             Display help and usage details
```
